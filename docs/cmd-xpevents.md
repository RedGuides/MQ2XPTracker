# /xpevents

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/xpevents [#]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
List the events and timestamps of the events. If a # is provided, it will list the exp events tracked in the last # seconds.
<!--cmd-desc-end-->
